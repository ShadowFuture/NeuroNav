# NeuroNav

A production-ready React starter for the NeuroNav app using Vite and TailwindCSS.

Quick start

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Project structure highlights

- `src/components` — shared UI components
- `src/pages` — route pages (15 categories)
- `src/layouts` — `MainLayout` with sidebar and topbar
- `src/context` — `AppContext` example
- `src/services` — API stubs for AI integration
- `src/styles` — Tailwind entry

Tailwind

Tailwind is configured in `tailwind.config.cjs` with a light-blue brand palette under `neuronav` colors.

This scaffold includes placeholder pages and fully implemented Daily Input, AI Brain Dump, and Weekly Review stubs.
